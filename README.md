# 장흥관산중학교 주간 시간표 디스플레이

구글시트(이번주 시간표 = 제일 왼쪽 시트)를 실시간으로 가져와 1920×1080 화면에 송출합니다.

## 동작 방식
- `/api/timetable` (Vercel 서버리스 함수)가 구글시트의 **제일 왼쪽 시트**를 CSV로 받아옵니다.
- `index.html`이 30초마다 이 데이터를 폴링해 시간표를 다시 그립니다.
- 따라서 구글시트에서 교과목을 수정하면 **30초 안에** 화면에 반영됩니다.

## 시트 연결
- 대상 시트 ID는 `api/timetable.js`의 `SHEET_ID` 또는 Vercel 환경변수 `SHEET_ID`로 지정합니다.
- 시트는 "링크가 있는 모든 사용자 - 뷰어"로 공유되어 있어야 합니다.

## 배포 (Vercel)
1. 이 폴더를 Vercel 프로젝트로 배포합니다.
2. 배포된 URL을 모니터 PC 브라우저에서 전체화면(F11)으로 띄웁니다.

## 모니터 표시 팁
- 크롬 키오스크 모드: `chrome.exe --kiosk --app=https://<배포주소>`
- 자동 새로고침은 앱이 알아서 합니다(별도 새로고침 불필요).
