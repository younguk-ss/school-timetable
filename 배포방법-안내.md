# 시간표 앱 직접 배포하기 (완전 초보용 · 터미널 없이 마우스만)

목표: 이 앱을 인터넷에 올려서 모니터 PC 브라우저로 주소만 입력하면 시간표가 뜨게 만들기.
방법: ① GitHub(파일 보관소)에 파일 올리기 → ② Vercel(무료 호스팅)에 연결해서 배포. 둘 다 무료.

준비물: 다운로드한 `timetable-app` 폴더(압축 풀어둔 상태), 학교 로고 이미지 파일.

------------------------------------------------------------
## 0단계. 로고 파일 이름 바꾸기 (선택)
- 학교 로고 이미지 파일 이름을 `logo.png` 로 바꿔서 `timetable-app` 폴더 안에 넣어 두세요.
  (jpg면 `logo.png`로 이름만 바꿔도 대부분 보입니다. 안 넣으면 임시 배지가 표시됩니다.)

------------------------------------------------------------
## 1단계. GitHub 가입
1. 인터넷 주소창에 github.com 입력 → 접속
2. 오른쪽 위 **Sign up**(가입) 클릭
3. 이메일·비밀번호·아이디 입력하고 안내대로 가입 (이메일 인증까지 완료)

## 2단계. 새 보관소(저장소) 만들기
1. 로그인 후 오른쪽 위 **+** 모양 → **New repository** 클릭
2. **Repository name** 칸에 `school-timetable` 입력
3. 아래 **Public**(공개)에 점 찍힌 상태 그대로 둠
4. 초록색 **Create repository** 버튼 클릭

## 3단계. 파일 올리기
1. 만들어진 화면에서 가운데쯤 **uploading an existing file**(파란 글씨) 클릭
   (안 보이면 **Add file → Upload files** 클릭)
2. 컴퓨터에서 `timetable-app` 폴더를 열고, 그 **안에 있는 파일/폴더 전부**
   (index.html, api 폴더, vercel.json, logo.png 등)를 마우스로 끌어다 업로드 칸에 놓기
   ※ 폴더(`timetable-app`)째로 말고, 그 **안의 내용물**을 올리세요.
3. 아래 초록색 **Commit changes** 버튼 클릭

## 4단계. Vercel 가입 (GitHub로 로그인)
1. 새 탭에서 vercel.com 접속
2. **Sign Up** → **Continue with GitHub** 클릭 → 권한 허용(Authorize)
   (GitHub 계정으로 바로 로그인되어 따로 비밀번호 안 만들어도 됩니다)

## 5단계. 배포하기
1. Vercel 첫 화면에서 **Add New… → Project** 클릭
2. 방금 만든 `school-timetable` 보관소가 목록에 보이면 옆의 **Import** 클릭
   (안 보이면 **Adjust GitHub App Permissions**로 접근 허용)
3. 설정은 아무것도 건드리지 말고 그대로, 아래 **Deploy** 버튼 클릭
4. 1~2분 기다리면 축하 화면과 함께 주소(예: `school-timetable.vercel.app`)가 나옵니다.

## 6단계. 모니터에 띄우기
1. 모니터 PC 크롬에서 그 주소 입력 → 시간표가 뜸
2. **F11** 키로 전체화면
3. 구글시트에서 교과목을 고치면 30초 안에 자동 반영됩니다(새로고침 안 해도 됨).

------------------------------------------------------------
## 매주 할 일
- 구글시트에서 **제일 왼쪽 탭**이 항상 "이번주" 시간표가 되도록만 해두면 됩니다.
  앱은 늘 제일 왼쪽 탭을 읽어 옵니다.

## 나중에 내용 고치고 싶을 때
- GitHub 보관소에서 그 파일을 열고 연필(✏️) 아이콘으로 수정 → Commit 하면
  Vercel이 자동으로 다시 배포합니다.

## 문제가 생기면
- 화면 오른쪽 아래에 "오프라인(미리보기)"라고 뜨면 시트를 못 불러온 것:
  구글시트가 "링크가 있는 모든 사용자 - 뷰어"로 공유돼 있는지 확인하세요.
